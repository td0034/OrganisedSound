# Results_4 analysis setup

## Setup
Create and activate a virtual environment, then install dependencies:

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
pip install -r req.txt
```

## Run (examples)
Auto-detects `data.json` or `sections_plus_transcriptions.json` in this folder:

```bash
python analyse_tz5_os_study_v2.py --outdir outputs
```

Specify an input file explicitly:

```bash
python analyse_tz5_os_study_v2.py --input data.json --outdir outputs --make-stats
```

Outputs are written to `outputs/tables` and `outputs/figures`. EPS, PNG, and TIFF
are generated by default; you can disable TIFF with `--no-tif`. Optional outputs
can be disabled with `--no-table2`, `--no-overload`, `--no-reliance`, and
`--no-param-salience`.
