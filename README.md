# Organised Sound Surveys

This repo contains survey apps, manuscript sources, and analysis outputs.

## Supplementary Materials
`Supplementary Materials/` contains the supplemental figures, tables, and supporting assets for the paper.

## Other top-level folders
- `Paper/` manuscript sources and results pipelines.
- `Participant Survey/` participant questionnaire app and exports.
- `Rater Survey/` blinded ratings app and exports.
- `Hardware.code-workspace` local VS Code workspace settings.
- `auto_survey_development_plan.md` internal planning notes.
